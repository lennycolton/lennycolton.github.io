Projects

VFPC

Flight Plan Checker

A flight plan checking plugin for EuroScope, written primarily in C++. This full-featured plugin gathers data from a REST API in JSON format, and confirms the validity of filed flight plans on the VATSIM Network, with detailed checking of a variety of important factors. I created the client-side element of VFPC, based upon an extremely simple, previous version, whilst a friend designed and built our bespoke REST API (currently closed-source).

VATGlasses

VATSIM User Map

A simple tool to display the location of different flights on the VATSIM network. Unique amongst the various mapping tools due to its highly accurate display of ATC sectors and coverage.

View on GitHub

VATSIM Hours Compiler

Hours Checker

A simple, yet powerful tool to check users' hours on various VATSIM ATC positions, with support for callsign variations and specific time periods. Gathers data from the VATSIM REST API in JSON format, and has the ability to save profiles as XML files.
